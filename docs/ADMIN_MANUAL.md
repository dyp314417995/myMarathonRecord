# 九州战马 · 管理员手册

---

## 一、管理入口

管理员首页在普通用户功能基础上，额外显示：

| 入口 | 说明 |
|------|------|
| 👥 用户管理 | 查看/搜索/排序所有用户，可删除用户 |
| ⭐ 积分管理 | 审批申请、录入积分、扣减积分 |
| 🔧 群组管理 | 管理跑群、上传二维码 |
| 🏁 赛事管理 | 添加/编辑/删除赛事 |
| 📋 跑群列表 | 查看各跑群信息 |
| 🏅 跑友名录 | 查看所有跑友名片 |

---

## 二、用户管理

- **搜索：** 输入即搜，全局匹配姓名/城市/群名，上限 100 条
- **排序：** 按姓名/10K PB/半马 PB/全马 PB/注册时间 ↑↓
- **详情：** 点击用户卡片 → 弹窗查看完整资料和积分余额
- **删除：** 点击「删除用户」→ 确认后用户数据清除，下次打开需重新注册
- **分页：** 每页 20 人，滑到底部自动加载更多

---

## 三、积分管理

### 审批申请

1. 进入「审批」标签
2. 查看待审批工单（含图片、时间、单号）
3. 点击「通过」或「拒绝」
4. 拒绝需填写原因
5. 通过后积分自动加到用户账户

### 录入积分（直接操作，无需审批）

1. 进入「录入/扣减」标签
2. 输入积分数量（正数=加分，负数=扣分）
3. 选择用户（支持多选）
4. 可选上传图片和填写原因
5. 点击「确认提交」，立即生效

> 选多人后名单超过 5 位会折叠，点「展开」查看全部。

---

## 四、赛事管理

**入口：** 首页「管理」→ 赛事管理

- **添加赛事：** 点「+ 添加赛事」→ 填写名称、日期、城市、类型、等级 → 保存
- **编辑/删除：** 点 ✏️ 修改，🗑 删除
- **状态自动更新：** 每天凌晨云函数自动将已过期赛事标为「已举办」

## 五、评价管理

**用户评价查看：** 赛事详情页 → 管理员可见「🛡 管理评价」面板 → 点「加载」查看所有评价 → 🗑 删除恶意/无效评价。

---

## 六、跑马记录

管理员和普通用户一样使用跑马记录功能（入口在「我的」面板）。

---

## 七、常见问题

**Q: 管理员看不到审批图片？**
A: 已修复。图片通过云函数转换，不受存储权限限制。

**Q: 用户积分显示为 0？**
A: 旧数据请执行 fixPoints 云函数一次性修复。
