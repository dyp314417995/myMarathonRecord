# 九州战马 · 管理员手册

---

## 一、管理入口

管理员首页在普通用户功能基础上，额外显示：

| 入口 | 说明 |
|------|------|
| 👥 用户管理 | 查看/搜索/排序所有用户，可删除用户 |
| ⭐ 积分管理 | 审批申请、录入积分、扣减积分 |
| 🔧 群组管理 | 管理跑群、上传二维码 |
| 🏁 赛事管理 | 添加/编辑/删除赛事 |
| 📋 跑群列表 | 查看各跑群信息 |
| 🏅 跑友名录 | 查看所有跑友名片 |

---

## 二、用户管理

- **搜索：** 输入即搜，全局匹配姓名/城市/群名，上限 100 条
- **排序：** 按姓名/10K PB/半马 PB/全马 PB/注册时间 ↑↓
- **详情：** 点击用户卡片 → 弹窗查看完整资料和积分余额
- **删除：** 点击「删除用户」→ 确认后用户数据清除，下次打开需重新注册
- **分页：** 每页 20 人，滑到底部自动加载更多

---

## 三、积分管理

### 审批申请

1. 进入「审批」标签
2. 查看待审批工单（含图片、时间、单号）
3. 点击「通过」或「拒绝」
4. 拒绝需填写原因
5. 通过后积分自动加到用户账户

### 录入积分（直接操作，无需审批）

1. 进入「录入/扣减」标签
2. 输入积分数量（正数=加分，负数=扣分）
3. 选择用户（支持多选）
4. 可选上传图片和填写原因
5. 点击「确认提交」，立即生效

> 选多人后名单超过 5 位会折叠，点「展开」查看全部。

## 四、赛事管理

**入口：** 首页「管理」→ 赛事管理

- **添加赛事：** 点「+ 添加赛事」→ 填写名称、日期、城市、类型、等级 → 保存
- **赛事类型：** 全马/半马可多选（混合赛事），10K/越野单选
- **赛事组：** 同赛事不同年份填同一个名称（如"北京马拉松"），评价自动跨年共享
- **编辑/删除：** 点 ✏️ 修改，🗑 删除
- **复制：** 点 📋 复制去年赛事，修改时间节点即可（赛事组自动继承）
- **状态自动更新：** 每天凌晨云函数自动将已过期赛事标为「已举办」

## 五、评价管理

- **赛事详情页** 可查看该赛事组所有用户的评分和评价
- **全马/半马** 评分单独计算，Tab 切换
- 跑友评价分页展示，每次 10 条

**已部署云函数：**
- `getRaceEvents`：分页查询赛事 + 评分聚合
- `getRaceReviews`：评价分页查询
- `reviewSummary`：异步 AI 总结（提交评价时触发）

## 七、常见问题

A: 已修复。图片通过云函数转换，不受存储权限限制。

## 六、赛事管理增强

### 发枪时间
- 鸣枪开跑支持多分区发枪：A/B/C... 区各自设定时间
- 添加分区字母自动递增，手动可改任意标识
- 时间默认上一区 +5 分钟

### 混合赛事
- 全马半马都选中时，规模和费用分别填写

### 积分规则
- 拉新/自媒体支持数量选择器
- 集体活动括号精简为（管理员录入）

## 七、活动管理

### 发布活动
首页 → 管理 → 活动管理 → 发布活动。
- 时间默认：开始 7:00，结束 +2h，报名截止 -1h，改时间自动联动
- 自定义报名字段：文本/单选/多选，上限 5 个
- 点击活动卡片查看详情和报名人员列表

### 管理操作
- 编辑 / 复制 / 取消 / 删除
- 自动完成：结束时间过后自动变为"已完成"并发放积分
- 状态筛选：横向滑动标签（报名中/进行中/已截止/已完成/已取消）
- 分页：每页 20 条，加载更多
- 默认显示今年活动，按最近时间节点排序

### 分享二维码
- 点击"分享"自动生成小程序码，长按保存
- 扫码跳转活动报名页

### 订阅通知配置
- 模板 ID：微信公众平台 → 订阅消息 → 选用模板
- AppSecret 存云函数环境变量 `APP_SECRET`
- 定时器 `checkRaceNotify` 每天 9:00 执行
- 向前端授权模板 ID 与云函数模板 ID 保持一致

**Q: 用户积分显示为 0？**
A: 旧数据请执行 fixPoints 云函数一次性修复。
