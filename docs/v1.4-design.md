## 一、跑马记录

### 1.1 首页入口

在首页「小工具」区新增卡片：

```
┌──────────────────┐
│  🏁              │
│  跑马记录         │
└──────────────────┘
```

### 1.2 记录列表（时光轴）

顶部 3 个筛选 Tab：**10K | 半马 | 全马**

每场比赛一张卡片，按时间倒序，卡片内容：

```
┌─────────────────────────────────┐
│ 🅰️ 全马 · 已完赛                  │
│                                 │
│ 2024 北京马拉松                   │
│ 日期：2024-11-03                 │
│ 城市：北京                       │
│ 成绩：3:28:15  🏆               │
│                                 │
│ ┌─────┐ ┌─────┐ ┌─────┐        │
│ │ 图1 │ │ 图2 │ │ 图3 │        │
│ └─────┘ └─────┘ └─────┘        │
│                                 │
│ "首马破330，太开心了！"           │
│                                  │
│ 🔓 已公开          ✏️ 编辑  🗑 删除│
└─────────────────────────────────┘
```

**空白状态：** 每个 Tab 下无记录时，显示空状态插画 + "暂无 10K 记录" + 「+ 添加记录」按钮。

### 1.3 添加/编辑记录表单

半屏弹出，分步填写：

```
┌──────────────────────────────┐
│  添加跑马记录              ✕  │
│                              │
│  赛事类型*     ○10K ○半马 ○全马│
│  赛事等级*     ○A  ○B  ○C    │
│  状态*         ○已报名          │
│               ○已中签          │
│               ○已完赛          │
│               ○未完赛(DNF)     │
│               ○弃赛(DNS)      │
│                              │
│  日期*        [年] [月] [日]  │
│  城市         [输入城市]       │
│  完赛成绩     [仅已完赛可填]   │
│  （小时:分钟:秒）              │
│                              │
│  图片         [+][+][+] 最多9张│
│                              │
│  参赛感言     [文本框]         │
│                              │
│  □ 公开（可在跑友名录查看）    │
│                              │
│       [取消]    [保存]        │
└──────────────────────────────┘
```

**状态联动：** 选「未完赛」或「弃赛」时，成绩输入框自动隐藏。

### 1.4 成绩变化（每个 Tab 独立显示）

每个 Tab 下方，如当前 Tab 是「全马」则只显示全马的成绩变化：

```
┌──────────────────────┐
│  📈 成绩变化    [展开] │
│  ▁▃▅▇▆▄▂▃            │
│  10K: 45:30 → 42:10  │
│  半马: 1:38 → 1:32   │
│  全马: 3:28 → 3:15   │
└──────────────────────┘
```

简单文字 + emoji 表示趋势，不画图表（审核风险 + 性能）。

### 1.5 跑友名录 → 个人详情

点开某个跑友的卡片，如果他有公开记录 + 隐私设置允许：

```
┌──────────────────────────────┐
│  张三 · 武汉 · 九州战队        │
│  ─────────────────────────── │
│  PB: 10K 42:10 │ 半 1:32 │ 全 3:15 │
│                              │
│  🏁 跑马记录（公开）           │
│  全马 · 2024北马 · 3:28:15    │
│  半马 · 2024汉马 · 1:35:20    │
│  10K · 2023光谷跑 · 45:30     │
└──────────────────────────────┘
```

## 二、隐私设置

### 2.1 入口

个人资料页底部 → 「隐私设置」

### 2.2 页面设计

```
┌──────────────────────────────┐
│  ← 隐私设置                   │
│                              │
│  个人信息                      │
│  ┌──────────────────────────┐ │
│  │  PB 成绩对外可见    [开关] │ │
│  │  关闭后其他跑友看不到你的PB │ │
│  └──────────────────────────┘ │
│                              │
│  预留                            │
│  ┌──────────────────────────┐ │
│  │  更多设置                 │ │
│  │  （未来扩展）             │ │
│  └──────────────────────────┘ │
└──────────────────────────────┘
```

仅控制 **PB 成绩** 是否公开。跑马记录的可见性由每条记录自己的「公开」开关决定。

## 三、数据存储

**collection: `race_records`**

| 字段 | 类型 | 说明 |
|------|------|------|
| `userId` | string | 关联用户 |
| `raceType` | string | '10k' / 'half' / 'full' |
| `raceLevel` | string | 'A' / 'B' / 'C' |
| `status` | string | 'registered' / 'won' / 'finished' / 'dnf' / 'dns' |
| `date` | string | '2024-11-03' |
| `city` | string | 城市 |
| `result` | string | 完赛时间，如 '3:28:15'，DNF/DNS 为空 |
| `images` | array | 图片 fileID 数组 |
| `note` | string | 参赛感言 |
| `isPublic` | bool | 是否公开 |
| `createTime` | date | |

**collection: `users` 新增字段**

| 字段 | 类型 | 默认 | 说明 |
|------|------|------|------|
| `privacy_pb` | bool | true | PB 是否公开 |
