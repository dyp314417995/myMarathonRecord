# 九州战马 · 我的跑马记录小程序 — 操作手册

---

## 一、注册与登录

| 步骤 | 操作 |
|------|------|
| 1 | 微信扫码或搜索打开小程序 |
| 2 | 新用户填写：头像、昵称、手机号、城市、10K/半马/全马 PB（选填）、选择所在跑群 |
| 3 | 点「确认注册」，注册即送 50 积分 |
| 4 | 老用户自动登录，无需重新填写 |

> 换手机不影响数据，积分/群组/记录都跟着微信号。

## 二、普通用户（所有注册用户默认角色）

### 首页功能

| 入口 | 说明 |
|------|------|
| 🏃 我的积分 | 查看积分余额、明细、申请积分 |
| 🏅 跑友名录 | 查看所有跑友的名片（昵称/城市/跑群/PB），支持搜索和排序 |
| 📋 跑群列表 | 查看各跑群信息，扫码进群 |
| 🤖 AI教练 | 跑步知识问答 + 深度思考 + 发图分析 |
| 📅 赛事日历 | （开发中） |
| ⏱️ 配速计算器 | （开发中） |

### 积分功能

**申请积分：** 我的积分 → 申请 → 选分类、填说明、上传图片（最多 9 张）→ 提交 → 管理员审批。

**查看明细：** 我的积分 → 可看到每一笔积分记录（赚取/扣减）、来源、时间。

### AI教练

**文字提问：** 输入问题（如"怎么提升配速"）→ 发送 → 知识库秒回。

**不满意？深度思考：** 点回复下方「🤖 深度思考」→ 调用 DeepSeek AI 深度回答。

**发图分析：** 点 📷 → 选择跑步截图/装备照 → AI 分析。

## 三、管理员（由超管配置）

### 首页功能：普通用户功能 + 以下管理入口

| 入口 | 说明 |
|------|------|
| 👥 用户管理 | 查看/搜索/排序所有用户，可删除用户 |
| ⭐ 积分管理 | 审批申请、录入积分、扣减积分 |

### 用户管理

- **搜索：** 输入姓名/城市/群名实时筛选
- **排序：** 按姓名/10K PB/半马 PB/全马 PB/注册时间 ↑↓
- **详情：** 点击用户卡片 → 弹窗查看完整资料和积分余额
- **删除：** 点击「删除用户」→ 确认后用户数据清除，下次打开需重新注册
- **分页：** 每页 20 人，滑到底部自动加载更多

### 积分管理

**审批申请：**
1. 进入「审批」标签
2. 查看待审批工单（含图片、时间、单号）
3. 点击「通过」或「拒绝」
4. 拒绝需填写原因

**录入积分（管理员直接操作，无需审批）：**
1. 进入「录入/扣减」标签
2. 输入积分数量（正数=加分，负数=扣分）
3. 选择用户（支持多选）
4. 可选上传图片和填写原因
5. 点击「确认提交」，立即生效

> 选多人后名单超过 5 位会折叠，点「展开」查看全部。

## 四、常见问题

**Q: 换手机数据还在吗？**
A: 在。数据存云端，绑微信号。

**Q: 积分申请多久生效？**
A: 管理员审批后即时生效。管理员自己提交的直接生效。
