# 九州战马 · 用户手册

---

## 一、注册与登录

| 步骤 | 操作 |
|------|------|
| 1 | 微信扫码或搜索打开小程序 |
| 2 | 新用户填写：头像、昵称、手机号、城市、10K/半马/全马 PB（选填）、选择所在跑群 |
| 3 | 点「确认注册」，注册即送 50 积分 |
| 4 | 老用户自动登录，无需重新填写 |

> 换手机不影响数据，积分/群组/记录都跟着微信号。

## 二、首页功能

### 我的

| 入口 | 说明 |
|------|------|
| 🏃 我的积分 | 查看积分余额、明细、申请积分 |
| 🏅 跑友名录 | 查看所有跑友名片（昵称/城市/跑群/PB），搜索排序 |

| 🏁 跑马记录 | 录入和管理个人跑马记录 |
| 🎉 活动管理 | （管理员）发布和管理活动 |

### 小工具

| 入口 | 说明 |
|------|------|
| 📅 赛事日历 | 查看赛事、标记参赛、评分评价 |
| 🎉 活动广场 | 浏览活动、报名参加、查看我的活动 |

### 编辑个人资料

点击左上角头像 → 编辑资料 → 可修改头像、昵称、手机号、城市、PB 成绩、所属跑群。

## 三、积分功能

**申请积分：** 我的积分 → 申请 → 选分类、填说明、上传图片（最多 9 张）→ 提交 → 管理员审批。

**查看明细：** 我的积分 → 可看到每一笔积分记录（赚取/扣减）、来源、时间。

---

## 四、赛事日历

**入口：** 首页「小工具」→ 赛事日历

**查看赛事：** 默认展示「我的赛事」。三个 Tab：
- 我的赛事：已标记想参加或已参加的赛事
- 所有赛事：全部赛事，支持搜索、标签筛选、按日期/评分/各维度排序
- 我的评价：已评价过的赛事，可查看和修改

**倒计时：** 未来赛事显示「距开赛 xx 天」，已办赛事显示「已举办 xx 天」。

**标记赛事：** 点击「标记我的赛事」→ 选择状态（计划报名/已报名/已中签/已完赛/未完赛/弃赛）。
- 混合赛事（全马+半马）：额外显示「参加项目」选择，默认全马
- 标记「已完赛」并填写成绩 → 自动同步到跑马记录，刷新 PB

### 赛事体验评分

**评分：** 赛事详情页 → 「⭐ 参与评分」→ 4 个维度打分（赛道/氛围/补给/交通，1-10 分）+ 标签选择 + 文字描述（200 字以内）。
- 混合赛事可选全马/半马
- 首次评分 +10 积分秒到账
- 每人每赛事限评一次，可修改
- 详情页「📝 我的评价」查看/编辑/删除

**赛事组共享：** 同一赛事不同年份（如 2025/2026 北京马拉松）评分互通。详情页看到的是整个赛事组的聚合数据。
- 全马/半马评分分开 Tab 展示
- **赛事总评**：系统自动总结跑友反馈
- **跑友评价**：分页展示他人评价（10 条/次），可按全马/半马筛选

**分享：** 赛事详情页右上角「…」→ 分享给跑友，扫码直接进入该赛事。

## 五、跑马记录

**入口：** 首页「我的」→ 跑马记录

**查看记录：** 顶部切换半马/全马/10K，每个 Tab 展示该类型的所有记录。

**添加记录：** 点击底部「+ 添加记录」→ 填信息 → 保存。

**填写注意：**
- 赛事类型由当前 Tab 自动确定
- 状态选「未完赛」或「弃赛」时无需填成绩
- 每条记录可设公开/私密
- 完赛且成绩优于历史最佳时，自动刷新 PB

**删除记录：** 点击卡片右下角 🗑 → 确认删除。

**越野跑特有：** 距离(km)、爬升(m)、赛事认证（ITRA/UTMB/UTMB WS）、完赛可选填 ITRA 表现分。

**他人记录：** 跑友名录 → 点用户 → 预览最近 3 条 → 「查看全部跑马记录」看完整版（只读）。

## 六、隐私设置

**入口：** 个人主页 → 隐私设置

- PB 成绩开关：关闭后其他跑友看不到你的 PB 数据
- 跑马记录的公开/私密在每条记录中单独设置

## 七、常见问题

**Q: 换手机数据还在吗？**
A: 在。数据存云端，绑微信号。

### 积分申请数量

- **拉新**：可选 1-10 人，一次申请多人，积分 = 3 × 人数
- **自媒体**：可选 1-4 篇，一次申请多篇，积分 = 3 × 篇数

## 七、活动广场

### 浏览活动
首页 → 小工具 → 活动广场，Tab 切换"所有活动 / 我的活动"。
- 状态标签：报名中 / 进行中 / 已截止 / 已完成 / 已取消
- 横向滑动筛选状态，默认显示今年活动
- 列表按最近时间节点排序，每页 20 条

### 报名
进入活动详情 → 填写自定义字段 → 立即报名。已报名"我的活动"中可取消。

### 分享
管理员可生成小程序码，扫码直达报名页。

## 八、赛事订阅通知

标记赛事时开启"倒计时提醒"，系统将在每个时间节点前一天 9:00 推送微信服务通知。
- 支持节点：报名开启/截止、出签、缴费截止、退费截止、候补、鸣枪开跑等
- 通知内容包含赛事名称、节点日期、事件类型

**Q: 积分申请多久生效？**
A: 管理员审批后即时生效。
