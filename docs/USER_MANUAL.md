# 九州战马 · 用户手册

---

## 一、注册与登录

| 步骤 | 操作 |
|------|------|
| 1 | 微信扫码或搜索打开小程序 |
| 2 | 新用户填写：头像、昵称、手机号、城市、10K/半马/全马 PB（选填）、选择所在跑群 |
| 3 | 点「确认注册」，注册即送 50 积分 |
| 4 | 老用户自动登录，无需重新填写 |

> 换手机不影响数据，积分/群组/记录都跟着微信号。

---

## 二、首页功能

| 入口 | 说明 |
|------|------|
| 🏃 我的积分 | 查看积分余额、明细、申请积分 |
| 🏅 跑友名录 | 查看所有跑友的名片（昵称/城市/跑群/PB），支持搜索和排序 |
| 📋 跑群列表 | 查看各跑群信息，扫码进群 |
| 🤖 AI教练 | 跑步知识问答 + 发图分析 |
| 📅 赛事日历 | （开发中） |
| ⏱️ 配速计算器 | （开发中） |

### 编辑个人资料

点击左上角头像 → 编辑资料 → 可修改头像、昵称、手机号、城市、PB 成绩、所属跑群。

---

## 三、积分功能

**申请积分：** 我的积分 → 申请 → 选分类、填说明、上传图片（最多 9 张）→ 提交 → 管理员审批。

**查看明细：** 我的积分 → 可看到每一笔积分记录（赚取/扣减）、来源、时间。

---

## 四、AI教练

**选择模型：** 顶部「千问 / DeepSeek」切换，回复标模型名。

**文字提问：** 输入问题（如"怎么提升配速"）→ 发送。

**发图分析：** 千问模式下点 📷 → 选择跑步截图/装备照 → AI 分析。DeepSeek 不支持图片。

---

## 五、常见问题

**Q: 换手机数据还在吗？**
A: 在。数据存云端，绑微信号。

**Q: 积分申请多久生效？**
A: 管理员审批后即时生效。
