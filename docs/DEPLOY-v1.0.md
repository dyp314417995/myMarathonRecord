## 1. 环境准备

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 用小程序管理员微信扫码登录开发者工具
- 克隆项目：`git clone https://github.com/dyp314417995/myMarathonRecord.git`
- 开发者工具中打开项目目录

---

## 2. 云开发配置

### 2.1 确认环境 ID

在 `miniprogram/app.js` 中确认：

```js
env: "cloud1-d5gy0iuiba5f9300f"
```

### 2.2 开通云开发

1. 点击开发者工具左上角「云开发」图标
2. 如未开通，按引导开通（选择"按量付费"，免费额度够用）
3. 确认环境名称和 ID 与配置一致

---

## 3. 云函数部署

在开发者工具左侧文件树中：

```
右键 cloudfunctions/getPhoneNumber  → 「上传并部署：云端安装依赖」
右键 cloudfunctions/initDB          → 「上传并部署：云端安装依赖」
```

验证：云开发控制台 → 云函数，状态显示「部署成功」。

---

## 4. 数据库配置

### 4.1 创建集合

云开发控制台 → 数据库 → 添加集合，依次创建：

| 集合名 | 说明 |
|--------|------|
| `users` | 用户表 |
| `admins` | 管理员配置表 |
| `groups` | 群组表 |
| `join_requests` | 加群申请表 |

### 4.2 安全规则

初期全部设为 `read: true, write: true`，上线稳定后再收紧。

### 4.3 初始化默认数据

`app.js` 首次启动自动创建「一群、二群、分舵群」。

或手动执行：
```js
wx.cloud.callFunction({ name: 'initDB' })
```

---

## 5. 设置首位超管

1. 真机预览小程序，完成注册
2. 云开发控制台 → 数据库 → `users` 集合
3. 修改 `role` 为 `"super_admin"`，`status` 为 `"approved"`
4. 重新打开小程序即见超管面板

## 8. 预览测试 Checklist

- [ ] 新用户注册 → 多选群 → 提交
- [ ] 个人资料查看/编辑/保存
- [ ] 超管配置管理员（有效期 + 权限）
- [ ] 管理员审批通过/拒绝
- [ ] 管理员移除用户
- [ ] 群组管理：增删改、排序、上传二维码

---

## 9. 提交审核

1. 开发者工具点击「上传」
2. 版本号填写 `1.0.0`
3. 公众平台 → 版本管理 → 提交审核
4. 等待 1-7 个工作日

---

## 10. 审核通过后

1. 版本管理 → 提交发布
2. 渐进发布：5% → 20% → 100%
3. 监控云函数日志
